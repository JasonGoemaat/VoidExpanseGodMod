# Super Gear Mod

This mod copies ships and gear from the game and gives them new names and stats.  To use
these in game, hit `t` to opent he chat window and type `/additem <item>`, replacing `<item>`
with these names:

* `hull_super`
* `device_mining_super`
* `device_emp_super`
* `engine_super`
* `fueltank_super`
* `generator_super`
* `grappler_super`
* `radar_super`
* `rcs_super`
* `shield_super`
* `weapon_laser_super`
* `weapon_autocannon_super`
* `weapon_mlrs_super`
